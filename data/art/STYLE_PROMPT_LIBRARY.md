# Style Prompt Library (Into the Woods)

Use these tags in art requisitions (`style_tags`) and prompt seeds.

## cozy_classic
- warm cabin browns, soft tungsten glow, restrained contrast
- readable silhouettes, low-noise painterly pixel base

## moonlit_mystery
- cool moon blues, wet bark highlights, deep shadow pockets
- mist layers, restrained saturation, nocturnal calm

## warm_hearth
- amber/orange firelight, rich wood grain warmth
- higher local contrast near light sources

## ink_noir
- high-contrast desaturation, graphite-to-silver tonal range
- noir silhouettes, dramatic edge lighting

## pastel_storybook
- soft desaturated pastels, airy haze, gentle light rolloff
- storybook calm, low aggression palette

## twilight_whispers
- muted purples and dusk teal, low-contrast fog depth
- melancholic evening tone with soft atmospheric bloom
